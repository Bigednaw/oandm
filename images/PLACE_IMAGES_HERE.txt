IMAGES FOLDER — O&M Professionals Website
==========================================

Drop all your image files in this folder.

REQUIRED IMAGES
---------------
HERO SLIDER (Home page):
  slide1.jpg   — 1920x1080px
  slide2.jpg   — 1920x1080px
  slide3.jpg   — 1920x1080px

ABOUT PAGE:
  about.jpg    — 800x1000px (portrait orientation)

PROJECT CARDS (projects.html):
Each project supports multiple images that auto-slide.

  Project 1: project1a.jpg, project1b.jpg, project1c.jpg
  Project 2: project2a.jpg, project2b.jpg
  Project 3: project3a.jpg, project3b.jpg, project3c.jpg
  Project 4: project4a.jpg, project4b.jpg
  Project 5: project5a.jpg, project5b.jpg, project5c.jpg
  Project 6: project6a.jpg, project6b.jpg

Recommended size for project images: 1200x675px (16:9 ratio)

To add more images to a project, add more files (project1d.jpg etc.)
and update the imgs[] array in projects.html.

ACCEPTED FORMATS: JPG, PNG, WebP

If an image is missing, the site will show a placeholder — it won't break.
